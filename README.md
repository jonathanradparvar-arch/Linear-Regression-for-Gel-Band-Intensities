The attached code solves the normal equation for linear (least sqaures) regression in the context of determining the relationship between gel band intensity of splicing products and arginine concentration in a project investigating arginine as a potential inhibitor of group I intron ribozymal splicing.

The analyzed gel bands, with changing arginine concentrations 0.0, 5.0, 10.0, and 17.5 mM, are shown in Panel A of the below image. The resulting linear regression model is depicted in Panel B.
